输入法

Rime

操作系统： Linux、Windows、Mac OS

是否开源/免费： 免费开源

官方网站： https://rime.im/

开源地址： https://github.com/rime

简述： Rime是一款简洁、高效、无广告的开源输入法。支持拼音、五笔、双拼、粤拼、繁体字等。在Linux下名称叫中州韵，Window下名称叫小狼毫，Mac OS下

截图软件

flameshot

操作系统： Linux

是否开源/免费： 免费开源

官方网站： https://flameshot.org/

开源地址： https://github.com/flameshot-org

简述： flameshot是一款Linux下的截图软件，类似于微信截图，可以拖拽，画线等，可以保存到本地或粘贴板

录屏软件

Kazam

操作系统： Linux

是否开源/免费： 免费开源

官方网站： https://launchpad.net/kazam

开源地址： https://code.launchpad.net/kazam

简述： Kazam是一款截图录屏组合的软件，支持录全屏、指定窗口、指定区域，声源支持系统声音、麦克风声音、组合声音等

文件搜索

everything

操作系统： window

是否开源/免费： 免费不开源

官方网站： https://www.voidtools.com/zh-cn/

简述： everything是一款window下的文件搜索软件，特点是简单、快。如果Linux下，可以使用 sudo updatedb + mlocate 进行搜索

图形绘制软件

yEd

操作系统： Linux、window、Mac OS

是否开源/免费： 免费不开源

官方网站： https://www.yworks.com/products/yed

简述： yEd是一款方便、快捷的图形绘制软件，与viso类似，可以绘制网络拓扑，人物关系，流程图等，体积比viso小

GIMP

操作系统： Linux、window、Mac OS

是否开源/免费： 免费开源

官方网站： https://www.gimp.org/

开源网站： https://gitlab.gnome.org/GNOME/gimp/

简述： GIMP是一款图形处理软件，类似于PS

视频编辑软件

Kdenlive

操作系统： Linux、window、Mac OS

是否开源/免费： 免费开源

官方网站： https://kdenlive.org/

开源网站： https://invent.kde.org/multimedia/kdenlive

机械设计软件

LibreCAD

操作系统： Linux、window、Mac OS

是否开源/免费： 免费开源

官方网站： https://www.librecad.org/

开源网站： https://github.com/LibreCAD/LibreCAD

简述： LibreCAD是一款制图软件，类似于autoCAD

FreeCAD

操作系统： Linux、window、Mac OS

是否开源/免费： 免费开源

官方网站： https://www.freecadweb.org/

开源网站： https://github.com/FreeCAD/

简述： FreeCAD是一款开源的参数化三维建模软件，可以符合基本的工程制图需求

电路设计软件

KiCAD

操作系统： Linux、window、Mac OS

是否开源/免费： 免费开源

官方网站： https://www.kicad.org/

开源网站： https://github.com/kicad

简述： KiCAD是一款开源的EDA软件，许多其他EDA软件在这上面都是以插件形式存在，由于开源，很多开发者提供了特别有意思的插件
